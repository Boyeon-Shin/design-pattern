# ☕ Java Design Patterns Study

Java 언어로 작성된 대표적인 디자인 패턴의 **개념 설명**과 **실습 예제**를 정리한 저장소입니다.  
GoF 디자인 패턴을 기반으로, 실무에서도 자주 사용되는 패턴을 중심으로 구성했습니다.

</br>

## 📁 패턴 분류   

#### ♢ 생성(Creational)
- [x] Singleton (싱글톤)
- [x] Factory Method (팩토리 메서드)
- [X] Abstract Factory (추상 팩토리)
- [ ] Builder (빌더)
- [ ] Prototype (프로토타입)

#### ♢ 구조(Structural)
- [x] Adapter (어댑터)
- [x] Decorator (데코레이터)
- [ ] Facade (퍼사드)
- [x] Composite (컴포짓)
- [ ] Proxy (프록시)

#### ♢ 행위(Behavioral)
- [x] Strategy (전략)
- [x] Observer (옵서버)
- [ ] Command (커맨드)
- [ ] State (상태)
- [x] Template Method (템플릿 메서드)
